README

To Compile: Open directory containing makefile, and use the make command in terminal.

To Run:
1: Squares and Diamonds
2: Spiral
3: Sierpinski Triangle
4: Barnsley Fern
5: Snowflake

Up: Increase iterations
Down: Decrease iterations
